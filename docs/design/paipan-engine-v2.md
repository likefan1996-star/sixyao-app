## 2. fortune.js v2 详细设计

### 2.1 月建旺相休囚死

**原理：** 月建（月令）主管当月五行旺衰。当令者旺，令生者相，生令者休，克令者囚，令克者死。

**春(寅卯月)：** 寅卯木旺(10), 巳午火相(6), 亥子水休(0), 申酉金囚(-6), 辰戌丑未土死(-10)
**夏(巳午月)：** 巳午火旺(10), 辰戌丑未土相(6), 寅卯木休(0), 亥子水囚(-6), 申酉金死(-10)
**秋(申酉月)：** 申酉金旺(10), 亥子水相(6), 辰戌丑未土休(0), 寅卯木囚(-6), 巳午火死(-10)
**冬(亥子月)：** 亥子水旺(10), 寅卯木相(6), 申酉金休(0), 辰戌丑未土囚(-6), 巳午火死(-10)
**四季月(辰/戌/丑/未)：** 辰戌丑未土旺(10), 申酉金相(6), 巳午火休(0), 寅卯木囚(-6), 亥子水死(-10)

**分值映射：** `旺=+10, 相=+6, 休=0, 囚=-6, 死=-10`

**输入：** 月建地支（从 dateutil.js 获取当前月份的月建）
**输出：** 每爻分值和文本描述

### 2.2 日辰十二宫生旺墓绝

**原理：** 日辰对爻的影响类似"当天状态"，用十二宫（长生→帝旺→死墓绝）判断。

简化版本 — 只取关键状态：

| 状态 | 含义 | 分值 |
|------|------|------|
| 长生/沐浴/冠带/临官/帝旺 | 生旺状态 | +8 |
| 衰/病 | 转弱 | 0 |
| 死/墓/绝 | 极弱 | -8 |
| 胎/养 | 萌芽 | +4 |

**五行长生十二宫表：**

| 地支 | 木 | 火 | 金 | 水土 |
|------|----|----|----|-----|
| 寅 | 临官 | 长生 | 病 | 病 |
| 卯 | 帝旺 | 沐浴 | 死 | 死 |
| 辰 | 衰 | 冠带 | 墓 | 墓 |
| 巳 | 病 | 临官 | 长生 | 绝 |
| 午 | 死 | 帝旺 | 沐浴 | 胎 |
| 未 | 墓 | 衰 | 冠带 | 养 |
| 申 | 绝 | 病 | 临官 | 长生 |
| 酉 | 胎 | 死 | 帝旺 | 沐浴 |
| 戌 | 养 | 墓 | 衰 | 冠带 |
| 亥 | 长生 | 绝 | 病 | 临官 |
| 子 | 沐浴 | 胎 | 病 | 帝旺 |
| 丑 | 冠带 | 养 | 墓 | 衰 |

注意：以爻的五行查日辰地支所在状态。例如爻为木，日辰为寅，则木在寅为"临官"（+8）。

### 2.3 用神自动匹配

**原理：** 根据用户输入的问题关键词，确定用神（核心关注点），然后判断用神的状态。

**匹配规则：**

| 分类 | 关键词 | 用神 |
|------|--------|------|
| 财运 | 财运, 收入, 赚钱, 投资, 生意, 买卖, 股票 | 妻财 |
| 官运 | 工作, 升职, 求职, 求职, 事业, 官非, 官司 | 官鬼 |
| 感情(男) | 婚姻, 女友, 老婆, 恋爱(男视角) | 妻财 |
| 感情(女) | 婚姻, 男友, 老公, 恋爱(女视角) | 官鬼 |
| 健康 | 健康, 疾病, 看病, 身体, 治疗 | 子孙 |
| 出行 | 出行, 旅行, 外出, 搬家 | 世爻 |
| 家宅 | 家宅, 房子, 装修, 风水 | 父母 |
| 考试 | 考试, 学业, 学习, 成绩 | 官鬼 |
| 默认 | (无匹配) | 世爻 |

**实现：** 在 `fortune.js` 内建关键词规则表，遍历匹配。

### 2.4 旬空判定

**原理：** 每旬有两个地支为空亡，空亡之爻暂时无用。

```
甲子旬: 戌亥空   甲戌旬: 申酉空   甲申旬: 午未空
甲午旬: 辰巳空   甲辰旬: 寅卯空   甲寅旬: 子丑空
```

**判定方法：** 从天干推算出所在旬，然后查表。

**分值影响：** 用神逢空 -15，其他爻逢空不参与生克

### 2.5 月破判定

**原理：** 月建所冲的地支即为月破。

```
子午冲  丑未冲  寅申冲  卯酉冲  辰戌冲  巳亥冲
```

**分值影响：** 用神月破 -20，等于完全无用

### 2.6 日冲判定

**原理：** 静爻逢日冲为"暗动"（+6），动爻逢日冲为"散"（-6）。

### 2.7 评分公式

```
总分 = 50 (基准)
 + 月建旺衰 (每爻平均, -10 ~ +10)
 + 日辰十二宫 (世爻/用神, -8 ~ +8)
 + 用神旬空 (-15, 若逢空)
 + 用神月破 (-20, 若月破)
 + 用神日暗动 (+6, 若静爻逢冲)
 + 动爻生扶用神 (+12/每个动爻)
 + 动爻克害用神 (-12/每个动爻)
 + 世应用神关系 (-10 ~ +10)
```

## 6. 边界情况和错误处理

- **无匹配关键词：** 默认用神 = 世爻，在结果页标注"默认以世爻为用神"
- **多关键词冲突：** 按优先级匹配（问财运同时问了工作→以第一个匹配为准）
- **用神不在卦中：** 伏神处理（先简化为"用神不现"，不深入伏神逻辑）
- **日辰计算：** dateutil.js 从系统时间取，如果用户跨时区用 UTC+8 硬编码
