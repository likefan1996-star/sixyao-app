# 六爻小程序

> 传统文化符号逻辑研究工具 — 微信小程序

基于《周易》六爻纳甲筮法构建的纯前端排盘工具，以"学术研究"定位过审微信小程序。

## 功能

- 📿 **六爻起卦** — 三枚铜钱，六次掷爻，还原古法
- 🧮 **自动排盘** — 纳甲、世应、六亲、六神一键生成
- 📊 **趋势评分** — 六档制逻辑同步率分析
- 📜 **古籍对照** — 64 卦卦辞爻辞即查即览
- 🔄 **变卦推演** — 自动识别动爻并生成变卦
- 🪙 **3D 铜钱** — SVG 金属质感 + 交错抛投动画
- 🔊 **拟真音效** — Web Audio 合成金属撞击声

## 技术栈

| 层 | 技术 |
|----|------|
| 框架 | 微信小程序原生 |
| 架构 | 纯前端（无后端） |
| 引擎 | JS 排盘引擎（纳甲/世应/六亲/六神） |
| 数据 | 64 卦 + 纳甲表 + 五行生克 |
| 样式 | Neo-Chinese Minimalism（新中式极简） |
| 合规 | 符号逻辑研究定位，去敏词汇 |

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/likefan1996-star/sixyao-app.git

# 2. 在微信开发者工具中导入 D:\sixyao
# 3. 开启"不校验合法域名"（开发期）
# 4. 编译运行
```
