# 周易六爻小程序

> 传统文化学习与自我梳理用的微信小程序排盘演示。

这是一个基于《周易》六爻排盘结构整理的纯前端小程序。项目重点不是宣称现实预测能力，而是把六爻中的本卦、变卦、纳甲、世应、六亲、六神、动爻等结构用更容易理解的方式展示出来，作为传统文化学习和自我梳理参考。

## 产品定位

- 用于传统文化学习、排盘结构演示和自我梳理。
- 不作为现实决策依据。
- 不使用「精准预测」「算命」「改运」「保证结果」等承诺性表达。
- 后续 AI 深度解读只负责整理和翻译既有盘面，不重新改写排盘结果。

## 当前功能

- 六爻起卦：支持长按铜钱逐爻起卦，也支持一键成卦。
- 自动排盘：生成本卦、动爻、变卦、纳甲、世应、六亲、六神。
- 结果页结论卡：首屏展示当前倾向、参考区间、白话说明和七档刻度。
- 专业盘面：保留六神、六亲、干支、世应和动爻标识。
- 术语小抄：把用神、动爻、世应、月日等术语翻译成白话。
- 结构依据：用「为什么这么看」帮助用户理解关键线索。
- 异常兜底：异常链接或非法参数不会直接崩页。
- 本地行为日志：关键动作通过轻量日志入口记录，后续可接入正式埋点。

## 重要说明

当前排盘引擎适合作为文化学习和结构演示的 MVP。基础排盘数据已有脚本审计，但解释评分仍是产品化模型，不是传统六爻的标准断法。

已知限制：

- 月建仍使用固定节气近似，节气交界日前后可能不准。
- 日辰旺衰、月破、日冲等判断仍是简化模型。
- 用神选择依赖关键词匹配，尚未结合用户角色和问事类型。
- 七档倾向是产品解释层，不能当作传统定论。
